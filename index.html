How we did it

The method, briefly

How the ranking works

  1. We collected every official draw for both games with a Python scraper — Powerball back to 1997, Mega Millions back to 2010.
  2. Both games have changed their number pools over the years (Powerball's whites went 49 → 53 → 55 → 59 → 69). Since a number can't be drawn before it exists, we count each number only over the draws it was actually in the pool for.
  3. For each number we compute draw rate = times drawn ÷ draws it was eligible for, then a luck score where 1.00 is exactly average.
  4. We show two views: the current game (headline, everyone equally eligible) and all-time (still by rate, so eras don't distort it).
  5. White balls and the special ball are always counted as separate pools. The retired Mega Ball 25 is shown separately, never ranked against today's 1–24.

Scorecard, not strategy.  Lottery draws are random and past results never predict future ones. This is a record of what happened — entertainment, not a system.  Correct as of the latest draws: Powerball 2026-07-01, Mega Millions 2026-07-03.
